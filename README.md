# 🧬 Biotechnology Sales Performance Analytics

## 📌 Project Overview
This project is an end-to-end data analytics dashboard built to analyze commercial e-commerce sales for a simulated biotechnology firm. The objective was to transform raw transactional data into a client-ready, interactive dashboard that highlights key performance indicators (KPIs), regional sales efficiency, and product profit margins.

This project was completed as part of the Future Interns Data Science & Analytics internship task.

## 🛠️ Tools & Technologies Used
Python (Pandas, NumPy): Used for initial data generation, cleaning, chronological sorting, and structuring realistic pricing models for biotech lab equipment.
  Microsoft Power BI: Used for data visualization, DAX measure creation, and dashboard layout design.

## 📊 Dashboard Features
The interactive Power BI dashboard (`Biotech_Sales_Analysis.pbix`) includes:
Custom DAX Measures: Calculated fields for Total Revenue, Total Profit, and Profit Margin %.
  Dynamic Conditional Formatting: Bar charts automatically highlight regions/categories crossing the strategic revenue threshold.
Interactive Slicers: A user control panel to filter metrics by specific regions and timeframes.
AI Smart Narrative: Automated, text-based generation of key takeaways and data summaries.
Dark Theme UI:** High-contrast, professional visual design for optimal readability.

## 📁 Files in this Repository
 `Realistic_Biotech_Sales.csv` - The cleaned, structured dataset containing realistic pricing and margins for biotech equipment (e.g., Mass Spectrometers, Gene Sequencers, PCR Machines).
 `Biotech_Sales_Analysis.pbix` - The fully functional Power BI dashboard file.
 `Dashboard_Report.pdf` - A static PDF export of the final dashboard for quick viewing.
